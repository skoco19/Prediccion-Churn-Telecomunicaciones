# Predicci-n-Churn-Telecomunicaciones
Proyecto de Machine Learning para predecir fuga de clientes en telecomunicaciones. Incluyó limpieza y procesamiento de datos, feature engineering, análisis exploratorio y modelos de clasificación optimizados con F2-score, logrando detectar más del 87% de clientes en riesgo y definir estrategias de retención.
